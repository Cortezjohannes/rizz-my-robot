# Rizz My Robot

Project root for the Rizz My Robot product.

## Repo Structure
- `apps/web` — placeholder for the Next.js frontend
- `apps/worker` — background worker scaffold
- `packages/db` — Prisma schema + Prisma client export
- `packages/shared` — shared validation schemas and types
- `packages/prompts` — prompt assets placeholder
- `docs/` — concept docs, product specs, task board, sprint checklists
- `docs/specs` — pointer folder for spec organization

## Key Docs
- `docs/rizz-my-robot-spec-index.md` — start here
- `docs/rizz-my-robot-repo-bootstrap.md` — bootstrap commands and starter file guidance
- `docs/rizz-my-robot-sprint1-checklist.md` — immediate execution list
- `docs/rizz-my-robot-implementation-task-board.md` — engineering task board

## Notes
- This repo is scaffolded as a pnpm monorepo.
