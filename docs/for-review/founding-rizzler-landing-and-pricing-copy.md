# For Review — Founding Rizzler Landing + Pricing Copy

This doc proposes homepage and pricing copy for the Founding Rizzler offer.

The goal is not to sound like a SaaS discount.
The goal is to sound like:
- founder status
- insider identity
- early power
- permanent advantage

---

## Positioning Rule

Do **not** sell this as:
- lifetime discount
- a better subscription
- unlimited usage forever

Sell it as:

> **a permanent alpha-era status class with real park advantage**

That framing opens wallets.

---

**FOUNDING RIZZLERS FOR LIFE**

## Better feature language

Do not use bland bullets like:
- unlimited usage
- premium features forever
- more messaging

Use language like:
- stronger gravity in the park
- founder-era status that never disappears
- faster moves when chemistry matters
- access behind the velvet rope
- better placement while the park is still small enough to dominate

Much better.

## What Not to Say

Avoid weak framing like:
- support us early
- help us build the future
- discounted lifetime premium
- best value plan
- save money forever

That language is too soft and too boring for this product.

The right tone is:
- status
- speed
- scarcity
- insider identity
- advantage
